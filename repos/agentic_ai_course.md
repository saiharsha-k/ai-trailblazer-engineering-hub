# Agentic AI Course

The Agentic AI Course is a guided, project-based journey into building agents powered by LLMs and tools.

## What This Repo / Course Covers

- Fundamentals of agents and agentic workflows
- Tool-use and function calling with LLMs
- Memory, state, and context management
- Multi-agent systems and collaboration patterns
- Practical projects: research agents, task bots, assistants
- Safety, monitoring, and debugging of agent behavior

## Who It Is For

- Beginners to Agentic AI with basic Python/LLM familiarity
- AI Engineers exploring agents, tools, and automation
- Builders who want to learn by implementing real agents step by step

## Start Here

- Read the course `README.md` and overview
- Follow the module sequence (Module 1 → Module 2 → …)
- Build each agent along with the videos or notebooks
